# 🌸 DOMAIN_KNOWLEDGE.md — K-Poker

> 화투 게임의 규칙과 Balatro 영감의 조화

## 1. 화투 기본 (Hwatu Basics)
- **12개 월(月)**: 각 월당 4장, 총 48장 구성
- **매칭**: 같은 월의 카드를 매칭하여 획득

## 2. 고스톱 족보 (Traditional Yaku)
- **광(Bright)**: 오광(15점), 사광(4점), 삼광(3점, 비삼광 2점)
- **빨간 띠(Hongdan)**: 3점
- **파란 띠(Cheongdan)**: 3점
- **초단(Chodan)**: 3점
- **고도리(Godori)**: 5점 (2, 4, 8월의 새)
- **피(Junk)**: 10장부터 1점, 추가당 1점

## 3. Balatro 시너지 시스템 (The Synergetic Gambit)
- **Scoring**: `Score = Chips × Mult`
- **Chips**: 카드 획득 시 기본으로 쌓이는 값
- **Mult**: 족보 및 스킬 시너지를 통해 합산/곱산되는 배율
- **Chain**: 스킬 카드의 배치(좌→우)에 따라 가산(+Mult) 후 승산(×Mult)이 적용되어 기하급수적 성장 가능

## 4. 특수 규칙 (Special Rules)
- **뻑(Ppeok)**: 3장 쌓인 곳에 마지막 1장을 내면 모두 획득 (전통 방식의 재해석)
- **싹쓸이(Sweep)**: 필드 카드를 모두 가져올 시 Chips 보너스
- **박(Bak)**: 특정 조건(광박, 피박) 달성 시 최종 Mult 2배
- **흔들기(Shake)**: 같은 월 3장 보유 시 Mult 2배
